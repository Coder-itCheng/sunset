# sunset

服务端：`SpringBoot` + `SpringCloud` + `MyBatis` + `SpringSecurity` + `Jwt` +  `Redis` + `RabbitMQ` + `Swagger`

数据库：`MongoDB` + `MySQL`

配置文件：https://github.com/Coder-itCheng/config-center.git

* clone 项目

```sh
git clone https://github.com/Coder-itCheng/sunset.git
```

* 启动 service

```tex

...
```
